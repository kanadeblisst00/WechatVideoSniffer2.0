## 新版本

几乎每天都有因为视频号下载加我的粉丝，这波流量还是要留一下的，所以重新开发了一个新版本，下载链接：https://wwzy.lanzoue.com/b068vkaij 密码:gxs4

网盘里有两个版本，一个是兼容win7，另一个移除了监听只支持win10以上系统

![](/img/new.png)

开源旧版本的README请看: [README_old](/README_old.md)

#### 功能列表

1. 支持下载单个视频
2. 支持批量下载用户视频
3. 支持批量下载用户直播回放

使用教程请看: [新版视频号下载使用教程(支持批量)](https://mp.weixin.qq.com/s/ncVi941iO_q0s55oNIxuVg)
